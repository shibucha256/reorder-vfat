# reorder-vfat

Windows向けのVFAT/exFATディレクトリ順序を並べ替えるTUIツールです。

## 使い方

1. `reorder-vfat.exe` を起動します。
2. 矢印キーで項目を選択します。
3. `INS` / `DEL` でエントリの順序を移動します。
4. `w` でVFAT順序の書き込みを実行します（確認あり）。

主要キー操作は画面右側のヘルプを参照してください。

## 注意

- 取り外し可能ドライブのみ対象です。
- FAT/exFAT以外のドライブには書き込みできません。

## ビルド

```
cargo build --release
```
